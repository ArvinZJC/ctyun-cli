# Changelog

## 0.1.0-beta.2 - 2026-07-21

### Changed

- Replaced portal-oriented and mixed-language generated labels with concise source-backed Simplified Chinese table labels.
- Raised the required core range to `>=0.4.0 <1.0.0` because typed request-body options rely on non-string JSON serialization introduced in core 0.4.0.

## 0.1.0-beta.1 - 2026-07-17

- Added the initial generated Cloud Disaster Recovery plugin with all 31 first-class-node APIs published by the selected upstream documentation version.
- Covered service activation, protection groups, protected cloud servers, replication and reverse replication, drills, failover, recovery points, tasks, network monitoring, and client lifecycle operations.
- Preserved reviewed source and promoted baseline evidence, typed request inputs, profile-region resolution, official response fixtures, localized help and table metadata, and confirmation for state-changing commands.
